Why Contribute?

  • Your work impacts users all over the world
  • Gain valuable experience and mentorship
  • Support mozilla's mission of an open internet for all
  • If feels good!

Thimble is an open source project created by the Mozilla Foundation in parnership with CDOT at Seneca College. It was created to help people learn to create their own web projects using HTML, CSS & Javascript.
